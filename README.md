Project: ISO 15939 Measurement Process Simulator

Project Description:
This project is a Java Swing desktop application that simulates the 5 core steps of the ISO/IEC 15939 software measurement process standard. The application implements an MVC architecture and allows users to define quality dimensions, collect data, and analyze results using a step-by-step wizard interface.

Compilation and Execution Instructions:
1. Open your terminal or command prompt and navigate to the directory containing the .java files.
2. Compile all the source files by running the following command:
   javac *.java
3. Start the application by running the following command:
   java MeasurementSimulator
